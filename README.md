Welcome to my personal developer portfolio website repository! This website showcases my projects, certifications, technical skills, and experience as a Full Stack Web Developer. It is built using modern web design principles featuring a dark-themed glassmorphism UI, interactive canvas graphics, and smooth micro-animations.

🔗 **Live Portfolio Link**: [https://nitishkpathak.github.io/Nitish-Portfolio/](https://nitishkpathak.github.io/Nitish-Portfolio/)

---

## 🌟 Premium Features & Interactions

*   🌓 **Seamless Light/Dark Theme Switcher**: Features immediate client-side theme loading using a blocking script in the document `<head>` to prevent the flash of unstyled content (FOUC).
*   🌌 **Interactive Canvas Particle Background**: A custom physics-based HTML5 `<canvas>` network particle system that drifts naturally and pulls slightly toward your cursor.
*   鼠标 **Custom Glow Cursor Follower**: A responsive, circular cursor helper that scales and changes color when hovering over interactive links or buttons.
*   📈 **Dynamic Scroll Progress Bar**: A visual indicator at the top of the viewport tracking your scroll position.
*   🔍 **Project Filter Categories**: Smooth transition filtering showing only selected project groups (`All`, `MERN / Web`, `Java & SQL`, `Frontend`) using scale transitions.
*   ✉️ **EmailJS Contact Form Integration**: A fully functional contact form that sends messages directly to my email address.
*   🍞 **Glassmorphic Bootstrap Toasts**: Success/failure states for the contact form are reported via beautifully styled, animated glass toasts instead of standard browser alerts.

## 🛠️ Local Development

To run this project locally, clone the repository and open `index.html` in your web browser:

```bash
# Clone the repository
git clone https://github.com/nitishkpathak/Nitish-Portfolio.git

# Navigate to directory
cd Nitish-Portfolio

# Open the project (using VS Code Live Server or by double clicking index.html)
```
